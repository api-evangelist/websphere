This GA release of Open Liberty 26.0.0.6 addresses significant security vulnerabilities and includes important bug fixes to enhance stability and performance. It covers CVE fixes affecting the sipServlet and appSecurity features, along with nine notable bug fixes spanning featureUtility configuration, EJB runtime issues, encryption, Windows configuration loading, telemetry APIs, TCP channel race conditions, HTTP/2 quiesce failures, and Windows service timeout handling.
