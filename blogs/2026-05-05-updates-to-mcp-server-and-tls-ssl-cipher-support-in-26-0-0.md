This beta release updates the mcpServer-1.0 feature and simplifies SSL cipher configuration by using the effective JDK cipher list by default and flexible enabledCiphers syntax. The Open Liberty 26.0.0.5-beta includes the following beta features (along with all GA features ): Updates to mcpServer-1.0 Update to TLS/SSL Cipher support See also previous Open Liberty beta blog posts . Updates to mcpServer-1.0 The Model Context Protocol (MCP) is an open standard that enables AI applications to access real-time information from external sources.
