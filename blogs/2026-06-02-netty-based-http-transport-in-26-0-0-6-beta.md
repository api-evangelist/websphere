Open Liberty 26.0.0.6-beta introduces a new Netty-based HTTP transport as an alternative to the existing HTTP channel framework, offering improved performance and modern non-blocking I/O capabilities. The post details how to enable the Netty transport, configuration options, and expected performance benefits for Liberty server deployments.
